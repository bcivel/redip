Saisie d'un incident

Vérification des derniers incidents

La page d'accueil de l'application liste les derniers incidents saisis. Vous commencez par contrôler si l'incident a déjà été saisi par un autre utilisateur.

  1. L'incident est déjà présent dans la liste.
  2. Le champ recherche permet de s'assurer que le ticket n'existe pas encore
  3. Après ces vérification, vous cliquez sur le bouton d'ajout d'incident

Formulaire d'ajout d'incident

La langue préférée pour la saisie des incident est l'anglais.

  1. Application Entrez ici le site concerné
  2. Severity Importance de l'incident, High : impact important sur les commandes.
  3. Start Le moment où l'incident à commencé, date du jour si le début est inconnu
  4. Email Entrez ici votre email, vous pouvez en mettre plusieurs séparés par des ;
  5. ProblemTile Entrez ici un titre court et explicite
  6. ProblemDescription Entrez ici le détail du problème rencontré, précisez le résultat attendu
  7. How to Reproduce Entrez les étapes permettant de reproduire le problème, précisez les produits ou comptes clients si nécessaire
